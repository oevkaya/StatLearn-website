Welcome to ADA 442 - Statistical Learning with R

This course introduces the introductory level of Statistical Learning with R course notes that are designed for 2021. Lecture slides are created by using the main course book and related readings. Weekly slides and hands-on session materials are all created by R programming. Main objective of this course is to introduce fundamental statistical modeling techniques to students and enable them to solve real life problems using these various methods using R language as a statistical computation tool.

Prerequisite: ADA 423 (TED University)


Course Goals

By the end of this course you will be able to:
  • classify and distinguish different learning methods (supervised or unsupervised, prediction or classification);
  • apply different prediction methods, interpret the results and outputs;
  • apply different classification methods, interpret the results and outputs;
  • solve real life problems using supervised and unsupervised statistical learning techniques by use of R programming language for learning algorithms;
  • write human- and machine-readable code using R.
